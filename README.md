# 邮箱实时监控

基于 **Vue 3 + Electron** 的桌面应用，输入邮箱和客户端密码后，可实时监控收件箱新邮件。

## 功能

- 支持常见邮箱：Gmail、QQ 邮箱、163、126、Outlook、Yahoo 等（IMAP）
- 使用**客户端密码 / 应用专用密码**登录（不使用账号登录密码）
- 定时轮询收件箱，新邮件自动出现在列表
- 点击邮件可查看发件人、主题、日期和正文

## 环境要求

- Node.js 18+
- npm 或 pnpm

## 安装与运行

```bash
# 安装依赖
npm install

# 开发模式（启动 Electron + 热更新）
npm run dev
```

## 客户端密码说明

| 邮箱     | 说明 |
|----------|------|
| **Gmail** | 在 Google 账号 → 安全性 → 两步验证 开启后，在「应用专用密码」中生成一个 16 位密码，填入本应用。 |
| **QQ 邮箱** | 设置 → 账户 → 开启 IMAP，并生成「授权码」，用授权码作为密码。 |
| **163/126** | 设置 → POP3/SMTP/IMAP → 开启 IMAP，并设置「客户端授权密码」。 |
| **Outlook** | 使用微软账号密码；若开启两步验证，需使用应用密码。 |

## 构建

```bash
npm run build
```

产物在 `release/` 目录。

## 技术栈

- **前端**: Vue 3、TypeScript、Vite
- **桌面**: Electron
- **邮件**: imapflow（IMAP）、mailparser（解析邮件正文）

## 安全提示

- 客户端密码仅用于本机与邮箱服务器之间的 IMAP 连接，不会上传到第三方。
- 建议使用各邮箱提供的「应用专用密码」或「授权码」，不要使用账号主密码。
